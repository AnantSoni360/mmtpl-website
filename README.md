## ✨ Key Features

- **🚀 Ultra-Fast Performance:** Built on the Next.js App Router with React Server Components and aggressive static generation.
- **🌗 Dynamic Theming:** Full support for pure high-contrast Light Mode and premium Dark Mode using custom CSS variables.
- **🗺️ Interactive Data Maps:** Custom-built interactive SVG maps for India and Overseas operations, complete with hover states, animations, and modal tooltips.
- **🎥 Cloudinary Video Integration:** Lag-free, memory-efficient background video streaming using raw HTML5 players served from the Cloudinary edge network.
- **✨ Fluid Animations:** Extensive use of `framer-motion` for stagger children effects, beautiful fade-ups, and smooth modal expansions.
- **📱 Fully Responsive:** Carefully crafted Tailwind classes ensure the UI looks flawless on massive desktop monitors and small mobile screens alike.

## 💻 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_GITHUB_USERNAME/mmtpl-website.git
   cd mmtpl-website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or yarn install / pnpm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add your database and API keys if required by Prisma.

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **View the site:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
